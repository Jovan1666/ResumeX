## 快速开始

### 启动应用

**一键启动（推荐）：** 双击项目根目录的 `启动.bat`（Windows）或 `start.command`（macOS），首次会自动安装依赖，完成后浏览器自动打开。

**手动启动：**

```bash
cd 界面
npm install    # 首次使用需要
npm run dev    # 启动开发服务器
```

启动后访问 http://localhost:5173

### 创建第一份简历

1. 点击首页的「开始制作」按钮
2. 在仪表盘点击「新建简历」
3. 进入编辑器开始编辑

## 编辑器使用

### 基本信息编辑

1. **上传头像**
   - 点击头像区域上传图片
   - 支持裁切功能（1:1 或 3:4 比例）
   - 建议使用正式证件照

2. **填写信息**
   - 带红色 * 的为必填项
   - 输入框会实时验证格式
   - 手机号需输入11位数字
   - 邮箱需符合标准格式

### 模块管理

1. **添加模块**
   - 点击「+ 添加模块」按钮
   - 选择预设类型或自定义

2. **编辑模块**
   - 点击模块展开/收起
   - 填写各项内容

3. **排序模块**
   - 拖拽模块左侧的 ⋮⋮ 图标
   - 调整模块显示顺序

4. **隐藏/删除模块**
   - 点击 👁 图标隐藏模块（保留数据）
   - 点击 🗑 图标删除模块

### 条目管理

每个模块内的条目（如工作经历）支持：

- **添加条目** - 点击「+ 添加」按钮
- **排序条目** - 使用 ↑↓ 按钮调整顺序
- **复制条目** - 快速复制相似内容
- **删除条目** - 移除不需要的条目

### 时间选择

- 点击时间输入框打开选择器
- 选择年份和月份
- 结束时间可选择「至今」

## 导出与打印

### 导出 PDF

1. 点击右上角「导出」按钮
2. 选择「导出 PDF」
3. 在打印对话框中选择「另存为 PDF」
4. 选择保存位置

**打印设置建议：**
- 纸张大小：A4
- 边距：无/最小
- 背景图形：勾选

### 导出图片

1. 点击「导出」→「导出图片」
2. 自动下载 PNG 格式高清图片

### 导出 Word

1. 点击「导出」→「导出 Word」
2. 自动下载 .docx 格式文档，可用 Word/WPS 二次编辑

### 样式调整

点击「样式」按钮可调整：

| 设置项 | 说明 |
|-------|------|
| 字体大小 | 85% ~ 110% 缩放，支持滑块精调 |
| 行间距 | 紧凑/标准/宽松/自定义（1.0~2.0 精确调节） |
| 页边距 | 窄/标准/宽/自定义（10~30mm 精确调节） |
| 字体风格 | 思源黑体/微软雅黑/黑体/思源宋体/楷体/仿宋/等宽 共 7 种 |

## 数据管理

### 自动保存

- 所有修改自动保存到浏览器本地存储
- 关闭页面后数据不会丢失
- 状态栏显示「已保存」表示数据已保存

### 导出备份

1. 进入仪表盘
2. 点击右上角「设置」图标
3. 选择「导出备份」
4. 保存 JSON 文件

### 导入备份

1. 点击「导入备份」
2. 选择之前导出的 JSON 文件
3. 确认导入（会覆盖现有数据）

### 清空数据

⚠️ **警告：此操作不可恢复**

1. 点击「清空所有数据」
2. 确认清空
3. 建议先导出备份

## 常见问题

### Q: 数据保存在哪里？

A: 数据保存在浏览器的 LocalStorage 中。清除浏览器数据会导致丢失，建议定期导出备份。

### Q: 如何在手机上使用？

A: 打开网页后，使用底部的「编辑」「预览」标签切换视图。建议在电脑上进行详细编辑。

### Q: 内容超出一页怎么办？

A: 
1. 系统会自动检测并显示警告
2. 点击「智能压缩」自动缩小字体
3. 或手动精简内容
4. 调整「样式」中的字体大小和行距

### Q: 如何提高简历评分？

A: 
- 完善所有基本信息
- 添加专业头像
- 个人总结控制在50-300字
- 工作/项目描述详细具体
- 使用量化数据（如：提升30%效率）

### Q: 打印出来是空白？

A: 
- 确保勾选「背景图形」选项
- 选择「无边距」或「最小边距」
- 使用 Chrome 浏览器效果最佳

### Q: 支持哪些浏览器？

A: 推荐使用以下浏览器：
- Google Chrome（推荐）
- Microsoft Edge
- Firefox
- Safari
